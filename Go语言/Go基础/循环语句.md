# 循环语句
***
>Go语言的循环语句更加宽松，但是只有For循环语句，抛弃了do/while以及while
>
> 示例中是各种类型的For循环语句
```Go
package main

import "fmt"

func main() {
    
    i := 1
    for i <= 3 {
        fmt.Println(i)
        i += 1
    }

    for j := 7; j <= 9; j ++ {
        fmt.Println(j)
    }

    for {
        fmt.Println("loop")
        break;
    }

    for n := 0; n <= 5; n ++ {
        if n%2 == 0 {
            continue
        }
        fmt.Println(n)
    }
}
```
